2
个人知识库 RAG 核心！
RAG = Retrieval-Augmented Generation（检索增强生成）
→ 让 AI 调用你自己的知识库回答，输出更准、更个性化
RAG 三大核心组件：
① 指令库 Prompt Library — 把用过的好 prompt 归档，分类复用
e.g. "扮演品牌总监"、"以表格输出"、"风格偏商务简洁"
② 核心信息库 — 项目成果、业务数据、历史对话记录
要清洗！去掉页眉页脚、图片水印等噪声
③ 信息标准化 — 文件格式优先级：
Markdown > PDF > Word > PPT > 图片 > 视频
越干净越好，减少噪声 = 提升AI理解效率
RAG 工作流程：
用户提问
→
检索文档
→
文本切块
→
向量化
→
匹配信息
→
注入上下文
→
生成回答
⚠ 拆分+向量化最耗算力 → 数据要精简
工具对比：
工具 特点 适合谁
